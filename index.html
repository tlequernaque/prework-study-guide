HTML

  • The head element contains information about a webpage
  • The body element represens the visible content shown to the user
  • The meta element contains information about the page that is used by the browser
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The script element embeds client-side scripts in the HTML document
  • The title element defines the title for a webpage, this element is required and is extremely important for Search Engine Optimization (SEO)

CSS

  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside an element
  • text-align:center is one way to center a title or heading on a webpage
  • img is an image selector that assigns property values to any image element in the markup
  • styly attributes to change of color of a single heading
  • card designates a class selector, any HTML element assigned to the class card will gain these property assigments from the CSS

Git

  • git status: checks what branch we are currently
  • git checkout -b "branch name": creates a new branch and switches to it
  • git checkout main: to switch to main branch
  • git add -A: to add the started code to the current working branch
  • git commit -m "add starter code": to commit with a message
  • git pull origin main: to double check if your local branch is in sync with the base branch in github
  • git push origin feature/starter-code: to push changes to the remote branch

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Array is a single variable that is used to hold a group of data
  • Index is a unique number of data items in an array
  • Function is simply a set of instructions that tells the computer how to perform a certain task
